# Control de Gastos Hormiga

¡Bienvenido! Este es un proyecto de Python diseñado para ayudarte a rastrear esos pequeños gastos diarios que parecen invisibles pero afectan tu bolsillo. 

El proyecto utiliza **Python** y la librería **Pandas** para organizar y analizar los datos de gastos, usuarios, comercios y categorías.
